Tugas 2 Diskritisasi#

Carilah data yang bertipe numerik (data klassifikasi)

  1. Lakukan proses diskritisasi dengan equal width dan equal frequency

  2. Lakukan proses diskritisasi dengan basis entropy

Diskritisasi juga disebut binning, yaitu merubah atribut numerik menjadi kategorikal. Digunakan untuk metode/model data mining yang tidak dapat menangani atribut numerik.

Dalam implementasinya, Diskritisasi memiliki dua pendekatan, equal-width intervals dan equal-frequency intervals.

Equal - Width Intervals Membagi rentang dari X menjadi k dengan lebar sama. Dengan cara yang sederhana yaitu membagi lebar rentang dari X dibagi k

Rumus : $\( \begin{align*} \displaystyle \text{w} &= \frac{x_{max}-x_{min}}{k} \end{align*} \)\( Dengan btas rentang ke-i yaitu \)\( \begin{align*} \displaystyle v_{i} &= x_{min} + iw \\ i &= 1,2, ... k-1 \end{align*} \)$

Equal - Frequency Intervals Membagi data menjadi k dengan Rentang frekuensi yang sama

Entropy Entropi adalah nilai informasi yang menyatakan ukuran ketidakpastian(impurity) dari attribut dari suatu kumpulan obyek data dalam satuan bit

Entropy

Entropy adalah

  • Rumus Entropy:

\[Entropy (D_{1}) = -\sum_{i=1}^{m} Pi \log_{2} Pi\]

  • Rumus Gain:

\[Gain(E_{new}) = E_{initial} - E_{new}\]

  • Rumus Info:

\[Info_{A}(D) = \frac{|D_{_{1}}|}{|D|} Entropy(D_{1}) + \frac{|D_{2}|}{|D|} Entropy(D_{2})\]

Fungsi yang digunakan untuk menghitung banyaknya data berdasarkan kategori

def overalCategory(sample, labels, col, category):
  group = sample.groupby(category).count()
  return [group.loc[label, col] for label in labels if label in group.index]

Fungsi yang digunakan untuk proses

def split(nilai, sample, labels, col, category):
  less_group = sample[sample[col] < nilai]
  greater_group = sample[sample[col] >= nilai]

  length_less_group = overalCategory(less_group, labels, col, category)
  length_greater_group = overalCategory(greater_group, labels, col, category)

  return (length_less_group, length_greater_group)

Fungsi untuk menghitung nilai entropy

# Fungsi Entropy 
def entropy(d):
  return -(sum([i/sum(d) * log2(i/sum(d)) if i/sum(d) != 0 else 0 for i in d]))

# Fungsi Info
def info(d, sample):
  return sum([(sum(i) / sample.shape[0]) * entropy(i) for i in d])

# Fungsi gain untuk menghitung selisih antara entropy awal dengan yang baru
def gain(inisial, new):
  return inisial - new
